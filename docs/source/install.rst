Install
=======

Install requirements
--------------------

The following pacakges are required::

    'numpy>=1.15.0'
    'pandas>=0.24.0'
    'scikit-learn>=0.22.1'
    'scipy>=1.1.0'
    'matplotlib>=3.0.0' 

Install from Github
--------------------

run the code below::

    pip install git+https://github.com/rogerlwlw/lwmlearn-rogerluo.git
